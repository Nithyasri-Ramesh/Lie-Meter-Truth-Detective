# Lie Meter: Truth Detective

A mobile-first mystery deduction game where the player reads chats, clues, and suspect statements to identify who is lying. Difficulty increases by level with more suspects, hidden clues, timers, clue penalties, emotional traps, and layered contradictions.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite.

## Build

```bash
npm run build
```

## Game features

- 30 playable cases
- Increasing difficulty tiers
- Truth / Lie / Suspicious decision system
- Confidence meter and score
- Hint system with penalties
- Streak, XP, rank, and case unlocks
- Local progress saving using localStorage
- Mobile-first responsive UI

## Convert to mobile app later

This is a React + Vite web game. You can deploy it as a PWA or wrap it using Capacitor for Android APK/AAB.
